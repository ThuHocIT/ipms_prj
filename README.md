# 🚀 Fullstack Web Application  
ReactJS + Spring Boot + MySQL

## 📌 Giới thiệu
Đây là project web fullstack gồm:
- **Frontend:** ReactJS
- **Backend:** Spring Boot (Java)
- **Database:** MySQL

Project được tổ chức theo mô hình **monorepo** (frontend & backend chung 1 repository).
